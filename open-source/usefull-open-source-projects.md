* **Docker Firefox:** Container Web Browser | [https://github.com/linuxserver/docker-firefox](https://github.com/linuxserver/docker-firefox)

```bash
docker run -d \
  --name=firefox \
  -e PUID=1000 \
  -e PGID=1000 \
  -e TZ=Etc/UTC \
  -p 3000:3000 \
  -p 3001:3001 \
  -v /DATA/AppData/firefox:/config \
  --shm-size="1gb" \
  --restart unless-stopped \
  lscr.io/linuxserver/firefox:latest
```
